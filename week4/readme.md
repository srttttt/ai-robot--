 常见误解：自由度 = 关节数                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  实际上：                                                   │
│  • 关节数 ≥ 自由度数（冗余）                               │
│  • 可以有"冗余自由度"（比如7轴机械臂 > 6个末端自由度）    │
│  • 冗余机械臂更灵活，可以避开障碍物                         │
│                                                   2D vs 3D 运动
维度	位置	姿态	总自由度
2D	x, y	θ	3
3D	x, y, z	roll, pitch, yaw	6
